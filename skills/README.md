## 🔍 搜索能力

### 语义搜索
- **描述**: 基于语义的搜索
- **技术**: 向量相似度、嵌入模型
- **应用场景**:
  - 文档检索
  - 相似性匹配
  - 推荐系统

### 全文搜索
- **描述**: 基于关键词的搜索
- **技术**: 倒排索引、BM25
- **应用场景**:
  - 文本搜索

  - 内容查找

- **描述**: 结合语义和关键词
- **优势**: 精准度和召回率平衡
- **应用场景**:
  - 智能检索
  - 搜索优化
  - 用户体验提升
