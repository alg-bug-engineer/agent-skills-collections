## 🌱 入门教程

适合初学者的基础教程，无需或只需要少量先修知识。

### [Agent Skills 初学者指南](./beginner-guides.md)
- **难度**: ⭐⭐☆☆☆
- **时长**: 2-3 小时
- **语言**: 中文
- **描述**: 零基础入门 Agent Skills 和 AI Agent 开发
- **内容**:
  - Agent 基本概念
  - 环境准备
  - 第一个 Agent
  - 核心组件介绍
- **适合**: 完全初学者

### LangChain 快速入门
- **难度**: ⭐⭐☆☆☆
- **时长**: 1-2 小时
- **来源**: [LangChain 官方文档](https://python.langchain.com/docs/get_started/introduction)
- **描述**: LangChain 框架的快速入门指南
- **内容**:
  - 安装和配置
  - 基础概念
  - 第一个 Chain
  - 简单的 Agent
- **适合**: Python 开发者

### AI Agent 基础概念
- **难度**: ⭐☆☆☆☆
- **时长**: 30 分钟
- **来源**: [OpenAI Documentation](https://platform.openai.com/docs/guides/function-calling)
- **描述**: 理解 AI Agent 的基本概念和工作原理
- **内容**:
  - 什么是 Agent
  - Agent 的组成
  - 工作流程
  - 应用场景
- **适合**: 所有学习者

## 🌿 进阶教程

需要一定基础知识的中级教程。

### LangChain 深入学习
- **难度**: ⭐⭐⭐☆☆
- **时长**: 5-6 小时
- **来源**: [LangChain Academy](https://academy.langchain.com/)
- **描述**: LangChain 框架的全面深入学习
- **内容**:
  - 高级 Chains
  - 自定义 Agents
  - 工具集成
  - 记忆系统
  - 向量存储
- **适合**: 有 LangChain 基础的开发者

### AutoGen 多智能体系统
- **难度**: ⭐⭐⭐⭐☆
- **时长**: 4-5 小时
- **来源**: [AutoGen Documentation](https://microsoft.github.io/autogen/docs/tutorial)
- **描述**: 学习构建多智能体协作系统
- **内容**:
  - AutoGen 核心概念
  - 智能体类型
  - 对话管理
  - 代码执行
  - 群聊机制
- **适合**: Python 开发者

### CrewAI 实战指南
- **难度**: ⭐⭐⭐☆☆
- **时长**: 3-4 小时
- **来源**: [CrewAI Examples](https://www.crewai.com/examples)
- **描述**: 使用 CrewAI 构建角色扮演智能体
- **内容**:

  - 执行流程
- **适合**: 想学习角色扮演的开发者

### Agent 技能开发
- **难度**: ⭐⭐⭐☆☆

## 🛠️ 实战项目

通过实际项目学习 Agent 开发。

### 项目 1: 智能问答助手
- **难度**: ⭐⭐☆☆☆
- **时长**: 3-4 小时
- **描述**: 构建一个能够回答用户问题的智能助手
- **技术栈**: LangChain, OpenAI API
- **功能**:
  - Web 搜索
  - 知识库查询
  - 对话记忆
  - 多轮对话
- **来源**: [LangChain Examples](https://github.com/langchain-ai/langchain/tree/master/cookbook)

### 项目 2: 代码审查 Agent
- **难度**: ⭐⭐⭐☆☆
- **时长**: 5-6 小时
- **描述**: 构建一个自动审查代码的 Agent
- **技术栈**: AutoGen, GitHub API
- **功能**:
  - 代码分析
  - 质量检查
  - 安全扫描
  - 改进建议
- **来源**: [AutoGen Examples](https://github.com/microsoft/autogen/tree/main/notebook)

### 项目 3: 研究助手 Agent
- **难度**: ⭐⭐⭐⭐☆
- **时长**: 6-8 小时
- **描述**: 构建一个辅助研究工作的智能助手
- **技术栈**: LangChain, AutoGen
- **功能**:
  - 文献检索
  - 内容总结
  - 数据分析
  - 报告生成
- **来源**: [Custom Project](../projects/README.md)

### 项目 4: 软件开发团队
- **难度**: ⭐⭐⭐⭐⭐
- **时长**: 10-12 小时
- **描述**: 模拟软件开发团队的多智能体系统
- **技术栈**: CrewAI, AutoGen
- **功能**:
  - 产品经理 Agent
  - 开发 Agent
  - 测试 Agent
  - 协作流程
- **来源**: [ChatDev](https://github.com/OpenBMB/ChatDev)

## 📚 学习路径

### 初学者路径（1-2个月）

**第1周**: 基础概念
- [Agent Skills 初学者指南](./beginner-guides.md)
- AI Agent 基础概念
- 环境搭建

**第2-3周**: 框架入门
- LangChain 快速入门
- 第一个 Agent 项目
- 简单的工具使用

**第4-6周**: 实践项目
- 智能问答助手
- 文件操作 Agent
- 基础的 Web 浏览

**第7-8周**: 进阶学习
- AutoGen 基础
- 多智能体概念
- CrewAI 入门

### 进阶者路径（2-3个月）

**第1-2周**: 框架深入
- LangChain 高级用法
- 自定义 Agent
- 记忆系统

**第3-5周**: 技能扩展
- Web 浏览技能
- 代码执行技能
- 数据库集成
- 工具链

**第6-8周**: 多智能体系统
- AutoGen 多智能体
- 对话管理
- 代码执行集成

**第9-10周**: 优化和部署
- 性能优化
- 错误处理
- 部署到生产

**第11-12周**: 复杂项目
- 代码审查 Agent
- 研究助手 Agent
- 自定义项目

### 高级路径（3-6个月）

**第1-2月**: 深入研究
- LangGraph 复杂工作流
- 多智能体优化
- Agent 架构设计

**第3-4月**: 企业级应用
- 生产级 Agent
- 大规模部署
- 监控和调试

**第5-6月**: 创新项目
- 自定义框架
- 新技能开发
- 论文复现
